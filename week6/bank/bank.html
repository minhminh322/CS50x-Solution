Week 6: Python

  1. Python Overview
  • Python là 1 ngôn ngữ bậc cao
  • Biến những thao tác phức tạp trên C trở nên dễ dàng hơn.
  • Ngôn ngữ phổ biến dùng, có tính ứng dụng rất cao như dùng trong viết web, khoa học dữ liệu, AI/machine learning, làm game, etc

  1. Python Syntax:
    • Để bắt đầu viết Python, tạo file đuôi .py
    • Ko giống như C program là 1 compiled language. Source code → machine code → Executable file.
    • Python là 1 interpreted language. Source code is executed line by line by python compiler.

  1. Variables
  • No type specifier
  • Declared by initializing only
  • No need semicolons

  • Printing and variable interpolation

  1. Conditionals
  • Similar to write English
  • && → And , || → Or
  • If elif
  1. Loops
  • While loop
  • For loop

  1. Arrays → Lists
  • Similar idea to array
  • Dynamic array:
    • Not fixed size
    • Can be grow or shrink
    • Add, remove elements in the middle

  • Append, remove
  • List comprehension

  1. Tuples
  • Not available in C
  • Iterable, Immutable

  1. Dictionaries
  • Similar concept of hash table in C
  • Key - value pair
  • Iterate using loop items()

  1. Functions
  • No return, and parameters data types.
  • main function
  • def keyword

  1. Struct → Objects
  • Define using class keyword
  • Constructor - initialization function (set starting value of the properties)
  • Self keyword - the first param of each method of object.
